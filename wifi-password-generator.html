What Is a WPA2 Password?

WPA2 (Wi-Fi Protected Access 2) is the security protocol used by the vast majority of home and office routers. When you set a WiFi password, you're setting the WPA2 Pre-Shared Key (PSK) — a passphrase that both your router and connecting devices use to derive a session encryption key.

WPA2 supports passwords from 8 to 63 characters. This generator always produces a 63-character password using the full printable ASCII set (letters, digits, and symbols), giving you approximately 413 bits of entropy — far more than any cracking hardware could ever approach.

If your router supports WPA3, use it. WPA3 uses SAE (Simultaneous Authentication of Equals) which makes offline dictionary attacks impossible even with a short password. But a long random WPA2 password is still excellent protection.

How to Set Your WiFi Password

1
Copy a password above — click the copy icon on any row.
2
Open your router admin panel — usually 192.168.1.1 or 192.168.0.1 in your browser. The default admin credentials are on a sticker on the router.
3
Go to Wireless → Security settings, set the mode to WPA2-Personal (or WPA3 if available), and paste your new password.
4
Save and reconnect your devices. Store the new password in your password manager — you'll need it when connecting new devices.

Why Your WiFi Password Matters

A weak WiFi password can be cracked by capturing the WPA2 4-way handshake (which happens whenever any device connects) and running it offline against a dictionary. This requires no brute force against the router itself — it's entirely local on the attacker's hardware.

  • Modern GPUs can test billions of guesses per second against captured WPA2 handshakes
  • Common passwords (password123, 12345678, street addresses) fall in seconds
  • A 63-char random password would take longer than the age of the universe to crack
  • WEP encryption is completely broken — if your router only supports WEP, replace it
  • Change your WiFi password when a housemate or employee with access leaves
