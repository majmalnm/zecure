How it works

Cryptographically secure by design

Most password generators use Math.random() — a pseudo-random function that is predictable and unsuitable for security. Zecure uses window.crypto.getRandomValues(), the same CSPRNG your operating system relies on.

Every password, key, and token is generated locally in your browser. There is no server. Disable your internet connection and reload — it works identically.

  • CSPRNG — Cryptographically Secure Pseudo-Random Number Generator
  • Rejection sampling eliminates modulo bias in character selection
  • No network requests, cookies, analytics, or server-side logging
  • Inspect the source yourself — open DevTools → Sources
  • Works offline after first load
entropy.js
// Browser CSPRNG — same source as your OS
function secureBytes(n) {
  const arr = new Uint8Array(n);
  crypto.getRandomValues(arr);
  return arr; // never leaves your browser ✓
}
